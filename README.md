## install redisearch ubuntu 20.04

```js 
$ apt update
$ apt install git
$ git clone --recursive https://github.com/RediSearch/RediSearch.git
cd RediSearch

$ apt install make
$ make setup
$ make build

$ ll build
redis-server --version
```
